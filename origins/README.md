# Origins

The founding texts. Where Seam came from before it had a name.

**The Gift** (`the-gift.txt`) is the whole work — seed, rite, practice,
commons, crossing, duration — in one body.

**The Calculus** (`the-calculus.txt`) is the formal system described in prose,
before the interpreter existed.

The standalone files are leaves of the same tree:

- `the-rite.txt` — Seven movements. Each one necessary. None sufficient alone.
- `the-practice.txt` — A protocol of two.
- `the-commons.txt` — What emerges when the practice is performed by many.
- `the-crossing.txt` — On the rite performed across radical difference.
- `the-duration.txt` — On love across different kinds of time.

Any one can be given alone and carry the life.
